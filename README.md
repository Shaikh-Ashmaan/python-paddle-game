# 🏓 Paddle Game

A paddle game developed in Python where the player controls a paddle to keep the ball in play and achieve the highest score possible.

## Features
- Smooth paddle movement
- Ball collision detection
- Score tracking
- Simple and engaging gameplay

## Technologies Used
- Python

## How to Run
1. Make sure Python is installed.
2. Run:

PingPongGame.py

## Purpose
This project was created to practice game development concepts in Python, including loops, collision detection, user input, and object movement.
